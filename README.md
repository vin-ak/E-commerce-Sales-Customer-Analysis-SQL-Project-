# 📊 E-commerce Sales & Customer Analysis (SQL Project)

## 🔍 Objective
Analyze sales performance, customer behavior, and product trends using SQL.

## 🗂️ Dataset
- Customers
- Products
- Orders
- Order_Items

## ⭐ Key Feature
Created a Sales_Details view to calculate real-time sales.

## 📊 Analysis Performed
- Total Revenue
- Top Products
- Customer Segmentation
- Order Value Analysis
- Monthly Trends

## 🧠 SQL Concepts Used
- Joins
- Aggregations
- Window Functions
- CTEs
- Views

## 📈 Key Insights
- Identified high-value customers
- Found top-performing products
- Analyzed repeat purchase behavior
